# 📝 To-Do List Website

A simple and responsive To-Do List web application built using **HTML**, **CSS**, and **JavaScript**. It allows users to add, delete, and manage their daily tasks directly from the browser.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete individual tasks
- Clear all tasks
- Tasks saved in browser using **localStorage**
- Responsive design for mobile and desktop


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- localStorage API
